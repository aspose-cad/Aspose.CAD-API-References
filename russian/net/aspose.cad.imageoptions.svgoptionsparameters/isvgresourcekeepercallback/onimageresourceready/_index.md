## ISvgResourceKeeperCallback.OnImageResourceReady method

Вызывается для каждого растрового изображения в SVG. Используйте его, чтобы указать, как хранить растровое изображение.

```csharp
public string OnImageResourceReady(byte[] imageData, SvgImageType imageType, 
    string suggestedFileName, ref bool useEmbeddedImage)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| imageData | Byte[] | Байты содержимого растрового изображения |
| imageType | SvgImageType | Тип изображения. |
| ПредлагаемоеИмяФайла | String | Имя предлагаемого файла. |
| useEmbeddedImage | Boolean& | если установлено` true` то изображение будет встроено в SVG. |

### Возвращаемое значение

Должен возвращать путь к сохраненному ресурсу. Путь будет использоваться в изображении SVG для ссылки на растровое содержимое. Путь должен указываться относительно целевого документа SVG.
