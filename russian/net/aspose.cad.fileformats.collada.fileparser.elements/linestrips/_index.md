## Linestrips class

LINESTRIPS. Элемент lineStrips предоставляет информацию, необходимую для связывания атрибутов вершин вместе, а затем организации этих вершин в соединенные полосы линий. Каждая линия-полоса, описываемая сеткой, имеет произвольное количество вершин. Каждый сегмент линии в полосе линий формируется из текущей вершины и предыдущей вершины.
