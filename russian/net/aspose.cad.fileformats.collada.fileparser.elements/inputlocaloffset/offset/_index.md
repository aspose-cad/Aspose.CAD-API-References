## InputLocalOffset.Offset property

Получает или задает смещение. Атрибут смещения представляет собой смещение в списке индексов. Если два элемента ввода имеют одинаковое смещение, они будут проиндексированы одинаково. Это работает как простая форма сжатия для списка индексов, а также определяет порядок, в котором должны использоваться входные данные. Обязательный атрибут.

public ulong Offset { get; set; }
