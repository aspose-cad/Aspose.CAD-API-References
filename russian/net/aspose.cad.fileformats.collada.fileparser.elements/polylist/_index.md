## Polylist class

ПОЛИСИСТ. Элемент POLYLIST предоставляет информацию, необходимую для связывания атрибутов вершин вместе, а затем организации этих вершин в отдельные полигоны. Многоугольники, описанные в POLYLIST, могут содержать произвольное количество вершин. В отличие от элемента полигонов, элемент POLYLIST не может содержать полигоны с дырками.

public class Polylist : ColladaElement
