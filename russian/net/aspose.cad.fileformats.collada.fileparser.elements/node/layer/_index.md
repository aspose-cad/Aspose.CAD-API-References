## Node.Layer property

Получает или устанавливает слой. Атрибут слоя указывает имена слоев, которым принадлежит этот узел. Например, значение "светящийся передний план" указывает, что этот узел принадлежит как слою "переднего плана", так и слою "светящегося". Значение по умолчанию пусто, что указывает на то, что узел не принадлежит ни к одному слою. Необязательный атрибут.

public string[] Layer { get; set; }
