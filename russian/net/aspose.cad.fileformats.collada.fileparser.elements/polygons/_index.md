## Polygons class

Полигоны. Элемент polygons предоставляет информацию, необходимую для связывания атрибутов вершин вместе, а затем организации этих вершин в отдельные полигоны. Описанные многоугольники могут содержать произвольное количество вершин. Эти полигоны могут быть самопересекающимися и содержать дыры.
