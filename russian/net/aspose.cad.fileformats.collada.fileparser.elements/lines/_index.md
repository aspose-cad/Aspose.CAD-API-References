## Lines class

Строки. Элемент lines предоставляет информацию, необходимую для связывания атрибутов вершин вместе, а затем для организации этих вершин в отдельные линии. Каждая линия, описываемая сеткой, имеет две вершины. Первая линия формируется из первой и второй вершин. Вторая линия формируется из третьей и четвертой вершин и так далее.

public class Lines : ColladaElement
