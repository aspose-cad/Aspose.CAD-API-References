## Brep.Source property

Получает или задает источник. Исходные элементы определяют доступ вершин, ребер и граней элементов к их геометрическим объектам. Для вершин необходим хотя бы один исходный элемент. Если есть ребра, необходим второй исходный элемент для доступа к кривым в элементе кривой с помощью массива IDREF_array. Если есть грани, третий исходный элемент необходим для доступа к поверхностям в элементе поверхности с помощью IDREF_array.

public Source[] Source { get; set; }
