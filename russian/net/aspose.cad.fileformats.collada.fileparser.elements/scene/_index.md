## Scene class

Сцена. Сцена воплощает в себе весь набор информации, которую можно визуализировать из содержимого ресурса COLLADA. Элемент сцены объявляет основу иерархии сцены или графа сцены. Сцена содержит элементы, которые составляют большую часть визуального и преобразующего информационного содержания, созданного с помощью инструментов разработки.
