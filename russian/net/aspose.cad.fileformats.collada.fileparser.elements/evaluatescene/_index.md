## EvaluateScene class

Сцена оценки. Элемент оценки_сцены объявляет информацию, определяющую, как оценивать эту визуальную_сцену. Может быть любое количество элементов Assessment_scene. Они оцениваются по порядку, и конкретный может быть отключен с помощью настройки enable = false.
