## Trifans class

ТРИФАНЫ. Элемент TRIFANS предоставляет информацию, необходимую для связывания атрибутов вершин вместе, а затем организации этих вершин в соединенные треугольники. Каждый треугольник, описываемый сеткой, имеет три вершины. Первый треугольник образован из первой, второй и третьей вершин. Каждый последующий треугольник формируется из текущей вершины, повторно используя первую и предыдущую вершины.

public class Trifans : ColladaElement
