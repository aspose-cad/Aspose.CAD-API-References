## Matrix.TypeGeneralTransform field

Эта константа указывает, что преобразование, определенное этим объектом выполняет произвольное преобразование входных координат. Если это преобразование может быть классифицировано любой из вышеуказанных констант, тип будет либо константой TypeIdentity, либо комбинацией соответствующих битов флага для различных координатных преобразований, которые выполняет это преобразование.
