## ColorPaletteHelper.GetCloseImagePalette method

Получает цветовую палитру из растрового изображения (палетизирует изображение), если у изображения его нет. Если палитра существует, она будет использоваться вместо выполнения вычислений.

```csharp
public static IColorPalette GetCloseImagePalette(RasterImage image, int entriesCount)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| image | RasterImage | Растровое изображение. |
| entryCount | Int32 | Количество требуемых записей. |

### Возвращаемое значение

Цветовая палитра, которая начинается с наиболее часто встречающихся цветов из*изображения*и содержит*Количествозаписей*записей.
