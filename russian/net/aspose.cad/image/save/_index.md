## Save(Stream, ImageOptionsBase) {#save_2}

Сохраняет данные изображения в указанный поток в указанном формате файла в соответствии с параметрами сохранения.

```csharp
public void Save(Stream stream, ImageOptionsBase optionsBase)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| поток | Stream | Поток для сохранения данных изображения. |
| optionsBase | ImageOptionsBase | Параметры сохранения. |

### Исключения

| исключение | условие |
| --- | --- |
| ArgumentNullException | optionsBase |
| ArgumentException | Не удается сохранить в указанный формат, так как в данный момент не поддерживается.; optionsBase |
| [ImageSaveException](../../../aspose.cad.cadexceptions/imagesaveexception) | Ошибка экспорта изображения. |

### Примеры

Экспортирует рисунок в формат JPEG, поворачивает его на 90 градусов и записывает в поток памяти

```csharp
using (var ms = new MemoryStream())
{
    using (var image = Aspose.CAD.Image.Load("fileName.dwg"))
    {
        image.Save(ms, new JpegOptions()
        {
            Rotation = RotateFlipType.Rotate90FlipNone
        });
    }
}
```
