## Load(string, LoadOptions) {#load_3}

Загружает новое изображение из указанного файла.

```csharp
public static Image Load(string filePath, LoadOptions loadOptions)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| filePath | String | Путь к файлу для загрузки изображения. |
| loadOptions | LoadOptions | Параметры загрузки. |

### Возвращаемое значение

Загруженный рисунок.

### Примеры

Загружает чертеж для обработки и выгружает все связанные ресурсы при вызове dispose

```csharp
using (var image = Aspose.CAD.Image.Load("fileName.dwg", new LoadOptions
{
    UnloadOnDispose = true
}))
{
    // обрабатываем рисунок
}
```
