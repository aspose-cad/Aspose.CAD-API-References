## GifOptions.PixelAspectRatio property

Получает или задает соотношение сторон пикселя GIF.

```csharp
public byte PixelAspectRatio { get; set; }
```

### Стоимость имущества

Соотношение сторон пикселя GIF.

### Примечания

Pixel Aspect Ratio - Коэффициент, используемый для вычисления аппроксимации аспекта соотношение пикселей в исходном изображении. Если значение поля не равно 0, это приближение соотношения сторон вычисляется по формуле: Соотношение сторон = (Pixel Aspect ) Ratio + 15) / 64 Pixel Aspect Ratio определяется как отношение ширины пикселя к его высоте. Диапазон значений в этом поле позволяет указать от самого широкого пикселя 4:1 до самого высокого пикселя 1:4 с шагом 1/64. Значения: 0 - Информация о соотношении сторон не предоставляется. 1..255 - Значение, используемое при вычислении.
