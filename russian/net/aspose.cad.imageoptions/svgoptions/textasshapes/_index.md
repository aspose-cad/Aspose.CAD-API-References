## SvgOptions.TextAsShapes property

Получает или задает значение, указывающее, должен ли текст преобразовываться в фигуры. По умолчанию текст будет преобразован в фигуры.

```csharp
public bool TextAsShapes { get; set; }
```

### Стоимость имущества

` true` если весь текст преобразован в формы SVG в преобразование; в противном случае` false` .
