## GifOptions.DoPaletteCorrection property

获取或设置指示是否应用调色板校正的值。

```csharp
public bool DoPaletteCorrection { get; set; }
```

### 适当的价值

` true` 如果应用调色板校正；否则，` false` 。

### 评论

调色板校正意味着每当图像导出为 GIF 时，都会分析源图像颜色 以便构建最佳匹配调色板（如果图像调色板不存在或未在选项中指定）。 分析过程需要一些时间，但输出图像将具有最佳匹配的调色板，并且结果在视觉上更好。
