## GifOptions.ColorResolution property

获取或设置 GIF 颜色分辨率。

```csharp
public byte ColorResolution { get; set; }
```

### 适当的价值

颜色分辨率。

### 评论

颜色分辨率 - 每种原色可用的位数 到原始图像，减 1。此值表示 从中选择图形颜色的整个调色板 的大小，而不是图形中实际使用的颜色数。 例如，如果此字段中的值为 3，则 原始图像的调色板具有可用于创建的每种原色 4 位 图片。该值应设置为指示 原始调色板的丰富程度，即使不是整个 调色板中的每种颜色都在源计算机上可用。
