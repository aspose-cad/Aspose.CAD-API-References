## Tristrips class

TRISTRIPS。 TRISTRIPS 元素提供了将顶点属性绑定在一起所需的信息，然后将这些顶点组织成相连的三角形。 网格描述的每个三角形都有三个顶点。 第一个三角形由第一个、第二个和第三个顶点组成。每个后续三角形都由当前顶点组成，重复使用前两个顶点。

public class Tristrips : ColladaElement
