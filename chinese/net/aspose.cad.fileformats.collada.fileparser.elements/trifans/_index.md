## Trifans class

TRIFANS。 TRIFANS 元素提供了将顶点属性绑定在一起所需的信息，然后将这些顶点组织成相连的三角形。 网格描述的每个三角形都有三个顶点。 第一个三角形由第一个、第二个和第三个顶点组成。 每个后续三角形都是从当前顶点形成的，重复使用第一个和之前的顶点。

public class Trifans : ColladaElement
