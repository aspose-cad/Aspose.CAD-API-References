## CanLoad(string) {#canload_2}

判断是否可以从指定的文件路径加载图片。


| 范围 | 类型 | 描述 |
| --- | --- | --- |
| filePath | String | 文件路径。 |

### 返回值

` true` 如果可以从指定文件加载图像；否则，` false` 。

### 例子

检查是否可以加载图形

```csharp

if (Aspose.CAD.Image.CanLoad(fileName))
{
    using (Aspose.CAD.Image drawing = Aspose.CAD.Image.Load(fileName))
    {
        // 处理绘图
    }
}
```
